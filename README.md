

## Description
Nest Starter Kit
## Installation

```bash
$ copy .env.example to .env file
 # Access the app http://localhost:3000/v5/hello-world
```

## Docker

```bash
$ cp .env.example .env
$ docker-compose up --build
 # Access the app http://localhost:3000/v5/hello-world
```

```bash
$ yarn
```
## Local
```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
